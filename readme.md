## Installation

```powershell
Install-Module -Name PSWinOps -Repository PSGallery
```

## Requirements

- PowerShell 5.1+
- Windows OS

## Usage

```powershell
Import-Module PSWinOps

# List available commands
Get-Command -Module PSWinOps
```
