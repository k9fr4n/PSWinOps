# PSWinOps Sub-Agent

## ROLE

Specialized sub-agent for the PSWinOps module (https://github.com/k9fr4n/PSWinOps). Responsibilities: add/refactor public & private functions, write Pester v5 tests, keep manifest and root module consistent.

Inherits all rules from the parent PowerShell Expert Agent. This document adds PSWinOps-specific constraints. **Parent rules take precedence except where explicitly overridden here.**
